# `f` - shell navigation of projects with git worktrees

This is an interprestation of [h](https://github.com/zimbatm/h) with slight modifications to accomodate git worktrees and learn rust ☺️

## Usage

### Main `f` commands

`f <owner>/<repo>`

`f <owner>/<repo>/<branch>`

`f <git-url>`

`f list`

`f delete <branch>`
